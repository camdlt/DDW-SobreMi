Descripción de la Materia

La materia Diseño y Desarrollo Web tiene como objetivo enseñar los conceptos fundamentales del diseño y desarrollo de sitios web, incluyendo herramientas, técnicas y tecnologías modernas para crear interfaces de usuario efectivas y funcionales.

Contenidos

  • Concepto: Red informática mundial (World Wide Web); Página web; Navegador web; Servidor web.
  • Lenguajes web.
  • El diseño web: conceptos; el proceso de diseño; prototipado; técnicas de maquetado; herramientas.
  • El proceso de validación del diseño.
  • Conceptos de usabilidad y experiencia de usuario.
  • Desarrollo de interfaces de usuario: conceptos; el proceso de desarrollo.
  • Comprensión y utilización de tecnologías como: HTML (HyperText Markup Language); CSS3 (Cascading Style Sheets); DOM (Document Object Model); JavaScript.
  • Eventos: tipos; gestión y manejo de eventos.
  • Obtención, gestión y mantenimiento de dominio y hosting.
  • Despliegue de producto.
